# 🎵 Task 5: Deploy a Static Website Using GitHub Pages

## 📌 Project Overview

This project demonstrates the deployment of a static website using **GitHub Pages**. The website, **MelodyX**, is a responsive music streaming web application built with HTML, CSS, and JavaScript. The project is hosted online through GitHub Pages, making it accessible via a public URL.

## 🚀 Deployment Steps

1. Created a public GitHub repository.
2. Added the project files (HTML, CSS, JavaScript).
3. Initialized Git and connected the local repository to GitHub.
4. Committed and pushed the project files.
5. Enabled **GitHub Pages** from the repository settings.
6. Selected the **main** branch and **root** folder for deployment.
7. Waited for GitHub Pages to publish the website.
8. Verified the live website through the GitHub Pages URL.
